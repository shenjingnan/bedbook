# 故事生成示例

本文档包含故事生成的各类示例，帮助理解和使用故事生成技能。

## 1. 不同输入类型的示例

### 1.1 故事名称输入

**输入**：
```
/story-generator 小兔子找太阳
```

**生成结果**：
```yaml
---
title: 小兔子找太阳
age: 3-7岁
keywords:
  - 好奇心
  - 探索
  - 勇敢
  - 成长
  - 森林动物
  - 自然认知
author: glm-5
category: 儿童故事
---

小兔子住在森林深处，从来没有见过太阳。

"太阳是什么样的呀？"小兔子问妈妈。

妈妈笑着说："太阳是圆圆的，暖洋洋的，会把整个森林都照亮。"

小兔子决定去找太阳。它跳过小溪，穿过草地，来到一片空旷的地方。

忽然，一个红红的大圆球从地平线上升起来，把天空染成了金色。

"啊！我找到太阳了！"小兔子高兴地跳起来，"太阳真的圆圆的，暖洋洋的！"

从那以后，每天早上，小兔子都会到空地上迎接太阳。它知道了，太阳是森林最好的朋友。
```

### 1.2 成语输入

**输入**：
```
/story-generator 守株待兔 --age 5-7岁
```

**生成结果**：
```yaml
---
title: 守株待兔的新故事
age: 5-7岁
keywords:
  - 勤劳
  - 智慧
  - 成长
  - 道理
  - 森林动物
  - 品德教育
author: glm-5
category: 儿童故事
---

小猪嘟嘟在森林边种了一片菜地。

一天，嘟嘟正在田里浇水，突然"砰"的一声，一只野兔撞在了田边的大树上，晕了过去。

嘟嘟高兴极了："哇，不用干活就有兔子送上门来！"

从那以后，嘟嘟不再浇水、不再除草，整天守在大树旁边，等着更多的兔子撞上来。

一天过去了，没有兔子来。
两天过去了，还是没有兔子来。
一个星期过去了，嘟嘟连一只兔子也没等到。

这时候，邻居小熊路过，看着嘟嘟荒废的菜地，摇摇头说："嘟嘟，天上不会掉馅饼，只有勤劳才能有好收成啊！"

嘟嘟脸红了。它明白了，那个意外的兔子只是运气，不能当饭吃。

从那以后，嘟嘟又重新开始认真种地，再也不会傻傻等待不劳而获了。

这个故事告诉我们：只有付出努力，才能获得真正的收获。
```

### 1.3 故事大纲输入

**输入**：
```
/story-generator 一只胆小的刺猬想要交朋友，但总是害怕被拒绝，最后勇敢地走出了第一步
```

**生成结果**：
```yaml
---
title: 小刺猬交朋友
age: 3-7岁
keywords:
  - 友谊
  - 勇敢
  - 信任
  - 成长
  - 森林动物
  - 情商教育
author: glm-5
category: 儿童故事
---

小刺猬住在森林边的小洞穴里，它长了一身尖尖的刺，看起来不太友善的样子。

其实，小刺猬心里很想交朋友。可是，它总担心别的小动物会因为它的刺而不喜欢它。

一天下午，小刺猬躲在草丛里，看见小兔子和小松鼠在玩捉迷藏。

"我也想玩……"小刺猬小声地说，但声音太小，谁也没听到。

小刺猬鼓起勇气，慢慢地从草丛里走出来。

"那个……我可以一起玩吗？"小刺猬的声音有点发抖。

小兔子和小松鼠愣了一下，然后小兔子笑着说："当然可以！你的刺可以帮我们藏东西呢！"

小刺猬高兴极了，它第一次发现，原来自己的刺也可以变成优点。

那天下午，三个小伙伴玩得开心极了。小刺猬明白了，勇敢说出心里话，才能收获真正的友谊。
```

## 2. 不同年龄段的故事示例

### 2.1 0-3 岁故事

```yaml
---
title: 小鸭子嘎嘎
age: 0-3岁
keywords:
  - 睡前故事
  - 亲子互动
  - 简单认知
  - 温馨
  - 小动物
  - 声音
author: glm-5
category: 儿童故事
---

小鸭子，嘎嘎叫。
游来游去，嘎嘎嘎。

看见小鱼，嘎嘎嘎。
看见小虾，嘎嘎嘎。

妈妈来了，嘎嘎嘎。
宝宝真乖，嘎嘎嘎。

太阳落山啦，
小鸭子回家啦。

睡觉觉，嘎嘎嘎。
晚安，小鸭子。
```

**特点说明**：
- 句子极短，便于婴幼儿理解
- 大量使用叠词和拟声词
- 强调节奏感，适合朗读
- 无明显情节，侧重感官体验

---

### 2.2 3-5 岁故事

```yaml
---
title: 小熊的红帽子
age: 3-5岁
keywords:
  - 友谊
  - 互助
  - 分享
  - 小动物
  - 温暖
  - 简单道理
author: glm-5
category: 儿童故事
---

小熊有一顶漂亮的红帽子，是奶奶给它织的。

小熊可喜欢这顶帽子了，每天都戴着它出门玩。

一天，风呼呼地吹，小熊的红帽子被风吹跑了！

"哎呀，我的帽子！"小熊急得团团转。

小松鼠看见了，从树上跳下来帮忙找。
小兔子看见了，也跑过来帮忙找。

"找到了！找到了！"小松鼠在草丛里找到了红帽子。

小熊高兴地说："谢谢你们！我们一起玩吧！"

三个小伙伴在草地上开心地玩了一下午。小熊想：有朋友真好啊。
```

**特点说明**：
- 情节简单但有发展
- 有小问题和小解决
- 对话简单直接
- 强调友谊和互助

---

### 2.3 5-7 岁故事

```yaml
---
title: 小狐狸的魔法种子
age: 5-7岁
keywords:
  - 冒险
  - 成长
  - 勇气
  - 森林动物
  - 奇幻
  - 责任心
author: glm-5
category: 儿童故事
---

小狐狸在森林深处发现了一颗发着金光的种子。

"这是什么种子呢？"小狐狸好奇地捡起来。

这时候，一只老猫头鹰飞过来说："这是一颗魔法种子，只有用真心照顾它，它才会开出最美的花。"

小狐狸决定好好照顾这颗种子。它每天给种子浇水、松土，还给它唱歌听。

一天天过去，种子发芽了，长出了绿绿的叶子。

有一天，暴风雨来了！小狐狸怕小苗被淋坏，找来大叶子给它挡雨。雨水噼里啪啦地打在大叶子上，小狐狸全身都湿透了，但它一直守护着小苗。

终于，雨停了，太阳出来了。

就在这时，小苗上开出了一朵七彩的花！花瓣闪闪发光，比小狐狸见过的任何花都美丽。

老猫头鹰飞来说："看吧，只有真心付出，才能收获最美的礼物。"

小狐狸骄傲地看着七彩花，它明白了——用心去做一件事，是世界上最棒的魔法。
```

**特点说明**：
- 情节更丰富，有冲突和解决
- 有一定的描写性语言
- 故事长度适中（约500字）
- 强调坚持和责任心

## 3. 故事模板

### 3.1 基础模板

```yaml
---
title: [故事标题]
age: [年龄段]
keywords:
  - [关键词1]
  - [关键词2]
  - [关键词3]
  - [关键词4]
  - [关键词5]
author: [作者名]
category: 儿童故事
---

[故事正文]
```

### 3.2 模板变量说明

| 变量 | 说明 | 示例值 |
|------|------|--------|
| `[故事标题]` | 故事名称，简洁有意义 | 小兔子找太阳 |
| `[年龄段]` | 目标年龄段 | 0-3岁 / 3-5岁 / 5-7岁 / 3-7岁 |
| `[关键词1-8]` | 覆盖四个维度的关键词 | 勇敢、友谊、成长 |
| `[作者名]` | 故事作者 | AI / Deepseek / [自定义] |
| `[故事正文]` | 完整故事内容 | - |

## 4. 常见错误示例

### 4.1 关键词数量不足

```yaml
# ❌ 错误 - 关键词少于5个
keywords:
  - 勇敢
  - 友谊
```

```yaml
# ✅ 正确 - 关键词5-8个
keywords:
  - 勇敢
  - 友谊
  - 成长
  - 互助
  - 温暖
  - 森林动物
```

### 4.2 关键词维度单一

```yaml
# ❌ 错误 - 关键词都来自同一维度
keywords:
  - 勇敢
  - 善良
  - 诚实
  - 勤劳
  - 聪明
```

```yaml
# ✅ 正确 - 关键词覆盖多个维度
keywords:
  - 勇敢          # 主题
  - 温暖          # 情感
  - 森林动物      # 角色
  - 成长          # 主题
  - 情商教育      # 教育
  - 友谊          # 主题
```

### 4.3 年龄段与内容不匹配

```yaml
# ❌ 错误 - 标注0-3岁但内容复杂
age: 0-3岁
---
小狐狸思考了很久，终于明白了友谊的真谛。它决定用智慧和勇气去帮助朋友解决困难...
```

```yaml
# ✅ 正确 - 年龄段与语言风格匹配
age: 0-3岁
---
小狐狸，蹦蹦跳。
找朋友，哈哈笑。
找到啦，找到啦！
小兔子，快快来！
```

### 4.4 缺少教育意义

```yaml
# ❌ 错误 - 故事没有积极的教育意义
---
小狼总是欺负其他小动物，抢走它们的食物，大家都怕它。最后小狼成了森林里最厉害的动物。
```

```yaml
# ✅ 正确 - 故事有积极的教育意义
---
小狼总是欺负其他小动物。有一天，小狼遇到了危险，但没有人愿意帮它。小狼这才明白，只有友善对待别人，才能得到真正的朋友。
```

## 5. 故事创作技巧

### 5.1 开头技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **场景开头** | 描述故事发生的环境 | 森林里住着一只小老虎... |
| **问题开头** | 提出故事的核心问题 | 小兔子有个大问题：它不知道太阳长什么样。 |
| **对话开头** | 用对话引出故事 | "我也能飞吗？"小鸟问妈妈。 |
| **悬念开头** | 设置悬念吸引注意 | 谁也没想到，小刺猬会有这样的秘密。 |

### 5.2 结尾技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **感悟式** | 角色获得感悟 | 小熊明白了，有朋友真幸福。 |
| **开放式** | 留下想象空间 | 明天，小兔子又会发现什么呢？ |
| **总结式** | 点明故事主题 | 这个故事告诉我们... |
| **温馨式** | 营造温馨氛围 | 慢慢地，小狐狸进入了甜甜的梦乡。 |

### 5.3 角色塑造

- **主角色**：1-2个，性格鲜明
- **配角**：2-3个，各有特点
- **命名**：简单好记（如小熊、嘟嘟、斑斑）
- **性格**：积极正面，有可学习之处

## 快速检查清单

### 生成前检查

- [ ] 故事标题不与现有故事重复
- [ ] 明确目标年龄段
- [ ] 确定故事主题和教育意义

### 生成后检查

- [ ] 关键词数量：5-8个
- [ ] 关键词维度：覆盖情感/主题/角色/教育
- [ ] 语言风格：符合年龄段
- [ ] 故事长度：符合年龄段要求
- [ ] 故事结局：积极正面
- [ ] 教育意义：明确且有价值
